# `log`

Logging is intentionally tiny and friendly. Use it for status updates, debugging, and making scripts feel alive.

```fl
log info "Starting…"
log warn "Wi-Fi not configured"
log error "Something went wrong"
log debug "scan={scan}"
```

Levels:

- `log info`
- `log warn`
- `log error`
- `log debug`
