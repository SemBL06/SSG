# SSG
A SSG for documentation. Used for all my projects. 

## Start
Run
```Bash
python main.py
```
to start the wizard. 

### Content
Doesn't matter, everyhting gets copied. 
#### .md
Should have a heading e.g:
```yaml
---
title: Welcome
sitemap:
  lastmod: "YYYY-MM-DD"
  changefreq: "monthly"
  priority: "0.5"
---
```
Then the rest of the .md file. 

## Features
- Generates sitemap
- Generates robots.txt
- Supports all filetypes
- Code highlighting
